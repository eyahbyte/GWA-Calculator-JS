# General Weighted Average (GWA) Calculator
A simple, dynamic web application built to quickly calculate general weighted average (GWA). Users can dynamically add courses, input grades and units, and switch between light and dark themes.

## Live Demo
https://eyahbyte.github.io/GWA-Calculator-JS/

## Features
* Dynamic course input generation using JavaScript.
* Accurate weighted average calculation logic.
* Dark Mode toggle funcitonality.
* Clean, minimalist desigh.

## Technologies Used
* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **FontAwesome**
